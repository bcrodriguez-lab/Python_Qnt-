    Revisión de campañas programadas

    Intervalo con el que el sistema consulta las campañas del día en SQLite y evalúa si debe ejecutarlas. Se aplica de inmediato sin reiniciar la aplicación.

    Mínimo {{ limits.campaign_check_interval_seconds_min }} s, máximo {{ limits.campaign_check_interval_seconds_max }} s.

    Mensaje en consola

    Job independiente de la revisión de campañas. Escribe en la terminal Consultando campañas programadas... cada N segundos (por defecto 60 = 1 minuto). No ejecuta ni consulta campañas en base de datos.

    Mínimo {{ limits.console_message_interval_seconds_min }} s (1 minuto).

    Retención de archivos de log

    Tiempo que se conservan los archivos execution_log_backup_*.txt generados al rotar el log. Los más antiguos se eliminan automáticamente. El archivo activo execution_log.txt no se borra.

    Mínimo {{ limits.log_retention_hours_min }} h, máximo {{ limits.log_retention_hours_max }} h ({{ parameters.log_retention_hours // 24 }} días con el valor actual).
